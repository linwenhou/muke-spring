软件版本

GA：General Availability，官方正式发布的稳定版本

同质的还有RELEASE,Stable，Final

RC：Release Candidate，发行候选版本

Alpha：内部测试版本，bug较多，功能不全

Beta：公开测试版本，比Alpha版本晚些，还会加功能，修bug

M：Milesttone，开发期发行版本，边开发边发行
